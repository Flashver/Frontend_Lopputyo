Softaa ajatakseen sinun tarvitsee kirjoittaa alhaalla oleva npm install, jotta se lataa
kaikki tarvittavat riippvuudet, jotta sovellus toimii.

npm install


ja ohjelmaa ajataksesi kirjoita terminaaliin komento:

npm run dev
